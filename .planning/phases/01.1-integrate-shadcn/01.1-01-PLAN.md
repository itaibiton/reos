<task type="auto">
  <name>Task 1: Update Shadcn configuration and regenerate components</name>
  <files>components.json, src/app/globals.css, src/components/ui/*.tsx, package.json</files>
  <action>
1. Update components.json to use Mira style:
   - Change "style" from "new-york" to "mira"
   - Change "iconLibrary" from "lucide" to "hugeicons"
   - Keep all other settings (rsc, tsx, tailwind config, aliases)

2. Install Inter font and Hugeicons:
   - Run: npm install @hugeicons/react-pro (or @hugeicons/react if pro not available)
   - Add Inter font via next/font/google in layout.tsx

3. Update CSS variables in globals.css for Mira style:
   - Mira uses smaller radius (--radius: 0.25rem for small)
   - Mira has tighter padding/spacing in components
   - Update CSS variable --radius to 0.25rem

4. Regenerate all Shadcn components with Mira style:
   - Run: npx shadcn@latest add button card input label --overwrite
   - This will pull components styled for Mira preset

5. Verify build passes: npm run build
  </action>
  <verify>
- components.json shows style: "mira" and iconLibrary: "hugeicons"
- package.json includes hugeicons dependency
- globals.css has --radius: 0.25rem
- npm run build succeeds without errors
  </verify>
  <done>
- Shadcn configured for Mira style with small radius
- Hugeicons installed
- All 4 UI components regenerated with Mira styling
- Build passes
  </done>
</task>

   - Menu icon → use Hugeicons menu equivalent (Menu01Icon or similar)
   - X icon → use Hugeicons cancel/close equivalent (Cancel01Icon or similar)

2. Update Sidebar.tsx:

     - Handshake → Handshake01Icon or similar

3. Note: Hugeicons uses different naming convention. Search the package for equivalent icons:

   - If not, run: npm uninstall lucide-react

5. Verify the app renders correctly: npm run dev
