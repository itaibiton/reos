Create FeatureCards directory and components following the ValuePropsSection pattern with these key differences:

1. **iconMap.ts** - Type-safe icon mapping:
   ```typescript
   import { Search, GitBranch, MessageSquare, Sparkles, type LucideIcon } from "lucide-react";

   export const iconMap: Record<string, LucideIcon> = {
     search: Search,           // FEAT-01: Property Discovery
     dealFlow: GitBranch,      // FEAT-02: Deal Flow (workflow stages)
     communication: MessageSquare, // FEAT-03: Communication
     aiAssistant: Sparkles,    // FEAT-04: AI Assistant
   } as const;

   export type IconKey = keyof typeof iconMap;
   ```

2. **FeatureCard.tsx** - Interactive card with Framer Motion gestures:
   - Use `motion.article` (not div) for semantic HTML
   - Add `variants={fadeInUp}` for scroll entry animation
   - Add `whileHover={{ scale: 1.03, y: -4 }}` with 0.2s duration (ONLY if !shouldReduceMotion)
   - Add `whileTap={{ scale: 0.98 }}` for touch feedback (ONLY if !shouldReduceMotion)
   - Add `tabIndex={0}` and `role="article"` for keyboard accessibility
   - Add `focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-landing-primary` for focus indicator
   - Use `useReducedMotion()` from framer-motion to conditionally disable hover/tap animations
   - CSS shadow transition: `shadow-sm hover:shadow-lg transition-shadow duration-300`
   - Border hover: `hover:border-landing-primary/30 transition-colors duration-300`
   - Icon container: 64x64px (w-16 h-16), rounded-xl, bg-landing-primary/10
   - Icon: 32x32px (w-8 h-8), text-landing-primary, strokeWidth={2}

3. **FeatureCardsSection.tsx** - Grid layout:
   - Use SectionWrapper with `id="features"` (for hash navigation from navbar)
   - Use SectionHeader with centered title and subtitle
   - Responsive grid: `grid-cols-1 sm:grid-cols-2 lg:grid-cols-4 gap-6 lg:gap-8`
   - Use `t.raw("items")` with type assertion for features array
   - Map icons using `iconMap[feature.icon]` with fallback

4. **index.ts** - Module exports:
   ```typescript
   export { FeatureCardsSection } from "./FeatureCardsSection";
   export { FeatureCard } from "./FeatureCard";
   ```
