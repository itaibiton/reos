2. Run `npx convex dev` - this will:

   - Name it "reos"
   - Generate convex/_generated/ directory
   - Start the Convex dev server

3. Create ConvexClientProvider component at src/app/ConvexClientProvider.tsx:
   - "use client" directive
   - Import ConvexProvider and ConvexReactClient from "convex/react"
   - Get NEXT_PUBLIC_CONVEX_URL from process.env
   - Create client with new ConvexReactClient(convexUrl)
   - Wrap children in ConvexProvider

4. Wrap app in ConvexClientProvider in src/app/layout.tsx

5. Create minimal convex/schema.ts with empty schema export (placeholder for future tables)

Note: If `npx convex dev` requires login, it will open browser - this is expected. The command handles auth automatically.
