**Status:** SHIPPED 2026-01-19

## Overview

Transform REOS into a social real estate platform with a feed for sharing properties, service requests, and discussions. Enable likes, saves, shares, following users, and public profiles with activity streams. Make it the global all-in-one platform for real estate professionals and investors.

## Milestone Summary

**Key Decisions:**

- Phase 21: Single posts table with postType discriminator (property_listing, service_request, discussion)
- Phase 21: Denormalized counters (likeCount, commentCount, shareCount, saveCount) on post documents
- Phase 23: PostCard acts as dispatcher routing to type-specific cards via switch on postType
- Phase 24: EngagementFooter uses optimistic UI with local state synced via useEffect
- Phase 25: Feed page has two-level tab navigation (source: Global/Following + type filter)
- Phase 26: Profile type detection via stats !== null (service providers have stats)
- Phase 27: Hacker News time-decay formula for trending: (P-1) / (T+2)^G with gravity 1.8
- Phase 27: GlobalSearchBar positioned center in header with 300ms lodash.debounce
